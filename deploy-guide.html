🚀 福建非遗美食助手 - 云端部署指南

把应用部署到 Render.com 免费平台，永久在线，无需电脑开机

预计用时：10-15 分钟
1 注册 GitHub 账号并创建仓库

如果已有 GitHub 账号，直接登录跳到「创建仓库」。

  1. 打开 github.com/signup，填写用户名、邮箱、密码注册
  2. 验证邮箱后，点击右上角 ＋ → New repository
  3. 仓库名称填：fujian-food-ai
  4. 选择 Private（私有）或 Public（公开）均可
  5. 不要勾选 "Add a README file"
  6. 点击 Create repository
💡 创建后页面会显示推送代码的命令，先不要关页面，下一步要用。
2 推送代码到 GitHub

在电脑上打开 命令提示符（CMD） 或 PowerShell，依次执行以下命令：

注意：把 你的用户名 替换为你的 GitHub 用户名

cd C:\Users\hh\WorkBuddy\2026-07-17-11-02-09\fujian-food-ai git remote add origin https://github.com/你的用户名/fujian-food-ai.git git push -u origin main
⚠️ 首次推送时 Git 会弹出 GitHub 登录窗口，输入你的 GitHub 账号密码登录即可。
如果提示需要 Personal Access Token，去 GitHub → Settings → Developer settings → Personal access tokens → Generate new token，勾选 repo 权限，生成后用 Token 替代密码。

推送成功后，刷新 GitHub 仓库页面，应该能看到所有代码文件。

3 注册 Render.com 并创建服务
  1. 打开 render.com/register，点击 "GitHub" 用 GitHub 账号直接注册
  2. 授权 Render 访问你的 GitHub 仓库
  3. 进入 Dashboard 后，点击 New + → Blueprint
  4. 选择刚才推送的 fujian-food-ai 仓库
  5. Render 会自动识别 render.yaml 配置文件
  6. 在环境变量设置中，填入 API Key（见下方表格）

需要设置的环境变量：

ck_fsce77fc1g5c.tSttHT5Mdnuvz-6uwXBZSrylCIMmvCdbqjw1v7T7iT0
4 启动部署
  1. 填好环境变量后，点击 Apply 或 Create
  2. Render 会自动开始构建：
    npm install → npm run build → npm run build:server
  3. 构建完成后自动启动：node server-dist/index.js
  4. 等待状态变成 "Live"（绿色），表示部署成功
💡 首次构建大约需要 3-5 分钟。如果构建日志中有警告（warning）可以忽略，只要最后显示 "Deploy succeeded" 即可。
⚠️ Render 免费版会在 15 分钟无访问后自动休眠，首次访问会冷启动（约 30 秒延迟）。之后正常访问即可。
5 获取公网地址并分享
  1. 在 Render 服务页面顶部，找到形如 https://fujian-food-ai-xxxx.onrender.com 的地址
  2. 这就是你的公网访问地址，发给朋友即可使用
  3. 每次你 git push 到 GitHub，Render 会自动重新部署
💡 可以在 Render 的 Settings 页面自定义域名前缀（如改为 fujian-food.onrender.com）

✅ 部署完成后

应用永久在线，朋友随时可访问，你的电脑不需要开机

如遇到问题，可在 Render 的 "Logs" 页面查看运行日志排查

📋 已准备的部署文件：
• render.yaml — Render.com 自动部署配置
• scripts/postbuild-server.cjs — 构建后自动创建 CommonJS 配置
• package.json — 包含 start 和 build:server 脚本
• .gitignore — 排除 node_modules、.env 等敏感文件
